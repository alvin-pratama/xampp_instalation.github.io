Panduan Instalasi XAMPP di Linux

Ditulis oleh | Terakhir diperbarui: 2024

XAMPP adalah paket perangkat lunak gratis dan open-source yang menyediakan lingkungan server web lokal untuk pengembangan aplikasi web. Paket ini mencakup Apache, MySQL, PHP, dan Perl. Dalam panduan ini, kita akan membahas cara menginstal XAMPP di sistem operasi Linux (Ubuntu/Debian dan CentOS/RHEL).

Prasyarat

  • Sistem operasi Linux (Ubuntu, Debian, CentOS, RHEL, atau turunannya)
  • Akses root atau hak sudo
  • Koneksi internet

Langkah 1: Unduh XAMPP untuk Linux

Buka browser dan kunjungi situs resmi XAMPP:

https://www.apachefriends.org/download.html

Cari versi terbaru untuk Linux (biasanya berformat .run). Salin tautan unduhan.

Atau gunakan terminal untuk mengunduh dengan wget (contoh untuk versi terbaru):

wget https://downloadsapachefriends.global.ssl.fastly.net/8.2.1/xampp-linux-x64-8.2.1-0-installer.run

Langkah 2: Berikan Hak Eksekusi

Setelah file diunduh, berikan izin eksekusi:

chmod +x xampp-linux-x64-*.run

Langkah 3: Jalankan Installer

Jalankan installer dengan perintah:

sudo ./xampp-linux-x64-*.run
Catatan: Jika muncul pesan "Permission denied", pastikan Anda sudah memberikan hak eksekusi pada langkah sebelumnya.

Ikuti instruksi pada layar. Secara default, XAMPP akan diinstal di direktori /opt/lampp.

Langkah 4: Memulai XAMPP

Setelah instalasi selesai, mulai layanan XAMPP dengan perintah:

sudo /opt/lampp/lampp start

Output yang diharapkan:

Starting XAMPP for Linux 8.2.1-0...
XAMPP: Starting Apache...ok.
XAMPP: Starting MySQL...ok.
XAMPP: Starting ProFTPD...ok.

Langkah 5: Uji Instalasi

Buka browser dan kunjungi:

http://localhost

Jika muncul halaman selamat datang XAMPP, maka instalasi berhasil!

Langkah 6: Mengelola Layanan XAMPP

Beberapa perintah penting lainnya:

sudo /opt/lampp/lampp start     # Mulai semua layanan
sudo /opt/lampp/lampp stop      # Berhenti semua layanan
sudo /opt/lampp/lampp restart   # Restart semua layanan
sudo /opt/lampp/lampp status    # Cek status layanan

Langkah 7: Akses phpMyAdmin

Untuk mengelola database MySQL, buka:

http://localhost/phpmyadmin

Langkah 8: Konfigurasi Keamanan (Rekomendasi)

Untuk meningkatkan keamanan, jalankan skrip keamanan:

sudo /opt/lampp/lampp security

Anda akan diminta mengatur password untuk MySQL root, menghapus contoh file, dan membatasi akses ke phpMyAdmin.

Langkah 9: Membuat Shortcut (Opsional)

Agar mudah diakses, buat symlink ke direktori bin:

sudo ln -s /opt/lampp/lampp /usr/local/bin/lampp

Sekarang Anda bisa menjalankan perintah tanpa path lengkap:

sudo lampp start

Menonaktifkan XAMPP Saat Boot

XAMPP tidak otomatis jalan saat boot. Jika ingin dijalankan otomatis, buat service systemd:

sudo nano /etc/systemd/system/xampp.service

Tambahkan konten berikut:

[Unit]
Description=XAMPP Control Panel
After=multi-user.target

[Service]
Type=forking
ExecStart=/opt/lampp/lampp start
ExecStop=/opt/lampp/lampp stop
User=root

[Install]
WantedBy=multi-user.target

Lalu aktifkan:

sudo systemctl daemon-reload
sudo systemctl enable xampp

Penutup

Selamat! Anda telah berhasil menginstal dan mengkonfigurasi XAMPP di Linux. Sekarang Anda siap untuk mengembangkan aplikasi web lokal seperti WordPress, Laravel, atau CodeIgniter.

Jika Anda mengalami masalah, pastikan port 80 (Apache) dan 3306 (MySQL) tidak digunakan oleh aplikasi lain. Gunakan perintah:

sudo netstat -tlnp | grep :80
sudo netstat -tlnp | grep :3306
